# Project Brief

## Project Name
SUSU - Student Learning Dashboard

## Program Details
- **Program**: Kiberbiztonsági mérnöki (Cybersecurity Engineering)
- **Code**: BBLFKM
- **Level**: BA/BSc/BProf
- **Study Mode**: Levelező (Part-time/Correspondence)
- **Admission Term**: 2025/26/1
- **Student Status Start**: 2025-09-01
- **Duration**: 7 semesters
- **Field**: Műszaki (Engineering)

## Project Goal
Build a clean, modern, highly usable web application that helps cybersecurity engineering students manage their semesters, subjects, deadlines, classes, exams, notes, study materials, and curriculum prerequisites.

## Target Users
University students (specifically BBLFKM cybersecurity engineering) who need to:
- Track semester progress across multiple subjects
- Manage deadlines, exams, and assignments
- Organize study materials and notes
- Visualize curriculum dependencies
- Access AI-powered learning assistance

## Core Value Proposition
- Centralized dashboard for academic management
- Visual curriculum mapping with prerequisite tracking
- Integrated note-taking with text and drawing capabilities
- AI-assisted study modules for different subjects
- Clean, modern interface optimized for daily use

## Key Features
1. **Semester Management**: Select and track multiple semesters with automatic current semester highlighting
2. **Subject Dashboard**: Comprehensive subject cards with status, deadlines, grades, and dependencies
3. **Calendar View**: Monthly calendar limited to semester period with classes, exams, and deadlines
4. **Global Notes**: Independent note-taking area with text and drawing capabilities
5. **Subject Detail Pages**: Notes, Study (AI-powered), Classes, and Subject Description
6. **Curriculum Map**: Visual graph of entire program with prerequisites and dependencies
7. **File Management**: Upload and preview study materials (PDF, txt, md, images)

## Success Criteria
- Users can efficiently manage their academic semester
- Clear visual hierarchy makes information easy to scan
- Responsive design works on desktop and mobile
- Fast performance with mock data
- Foundation ready for future AI integration and persistence
